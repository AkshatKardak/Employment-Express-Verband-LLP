## 🎨 **Customization Options**

### 🎨 **Brand Colors**
Edit `assets/css/styles.css`:
```css
:root {
    --primary-color: #007bff;      /* Main brand color */
    --secondary-color: #20232a;    /* Secondary color */
    --accent-color: #f8f9fa;       /* Accent color */
}
```

### 📝 **Content Updates**
- **Admin Panel**: Update testimonials, banners, settings
- **Direct Editing**: Modify HTML files for structural changes
- **Pricing**: Update service packages in `assets/js/pricing.js`

### 🌐 **Custom Domain (Optional)**
1. Settings → Pages → Custom domain
2. Enter: `anitechcs.com`
3. DNS: Add CNAME record → `akshatkardak.github.io`
4. Enable "Enforce HTTPS"
